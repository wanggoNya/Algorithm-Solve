# Algorithm-Solve
알고리즘 문제 풀이 코드를 저장하는 레파지토리입니다.   
백준 사이트와 B조 스터디에서 받은 문제로 공부 중입니다.   
자바는 이클립스, 파이썬은 VSCODE 사용 중입니다. 
  
## 디렉토리 설명
* c_baekjoon
  * 사용 언어 : C
  * 백준 단계별 & 문제번호 풀이 저장
* python_baekjoon
